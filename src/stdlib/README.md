## Contents

This is the standard base library of the Lento programming language written in Rust. \
Including:

- Error handling
- I/O
- File system
- Networking
- Concurrency
- Threading
- Synchronization
- Time
- Date
- Math
- Random
- Cryptography
- Compression
- Encoding
